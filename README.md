# Production Performance & Quality Analytics Dashboard
# 🚀 Overview
This Power BI project provides a comprehensive view of operational performance, focusing on production volume, supplier efficiency, and quality assurance. By visualizing key metrics such as total transactions, quantity trends, and rejection rates, this dashboard enables stakeholders to identify production bottlenecks and monitor supplier reliability.

# 📊 Key Insights
Production Trends: Tracks monthly production volume to visualize performance over time.

Supplier Analysis: Compares total quantities supplied by different vendors to assess volume distribution.

Quality Control: Analyzes the relationship between production quantity and rejection rates to identify potential manufacturing issues.

KPI Tracking: High-level summary metrics (Total Transactions, Avg Quantity per Order, Total Quantity) for quick performance assessment.

# 🛠 Tech Stack
Tool: Microsoft Power BI Desktop

Data Source: Company data sheets

Visualization: Area charts, Bar charts, KPI cards, and Slicers.
